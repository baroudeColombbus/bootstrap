Bootstap 4.5 - Introduction

Cette page focntionne avec Bootstrap4 en CDN

Présentation générale

  • Bootstrap est un framework CSS "cadriciele"
  • C'est une une librairie côté client (front end)
  • C'est une une librairie CSS mais il y a également des composants HTML et JavaScript
  • Il comporte également une grille de mise en page

Intéret ?

  • Uniformatisation pour les navigateurs
  • Gain de temps
  • Styles identiques
  • Plein de suppléments en +
  • Grille responsive
  • Gratuit
  • Très bonne gestion des média queries
